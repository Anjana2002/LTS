# LTS Internship Projects

This repository contains projects developed during the **LTS Internship**, showcasing skills in **PHP, Symfony, React, MySQL, Redux/RTK Query, and TanStack**.

---

## Projects Overview

### 1. Ecommerce Website
**Description:**  
A simple e-commerce website for basic online shopping functionality.  

**Technologies:**  
PHP, MySQL (phpMyAdmin), HTML, CSS, jQuery  

**Features:**  
- Product listing and details  
- Shopping cart  
- Basic checkout functionality  

---

### 2. MovieMate
**Description:**  
A movie ticket booking system built using Symfony.  

**Technologies:**  
Symfony (PHP), MySQL (phpMyAdmin)  

**Features:**  
- User registration and login  
- Browse movies and showtimes  
- Book tickets and manage bookings  

---

### 3. Spiceleaf
**Description:**  
Beginner-friendly React project for forms and data display.  

**Technologies:**  
React, JavaScript, HTML, CSS  

**Features:**  
- User input forms  
- Display submitted data dynamically  

---

### 4. Quicknotes
**Description:**  
A React application that interacts with external APIs.  

**Technologies:**  
React, Axios, JSONPlaceholder API  

**Features:**  
- Fetch and display notes from API  
- Create, update, and delete notes  

---

### 5. Fullstack Project
**Description:**  
A React-based project using advanced state management and data handling.  

**Technologies:**  
React, Redux, RTK Query, TanStack, Symfony 

**Features:**  
- Add, edit, and delete data entries  
- View statistics and summaries  

---
